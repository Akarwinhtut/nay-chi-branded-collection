# Astra Meridian Hotel

Standalone hotel landing page built with Next.js App Router and Tailwind CSS.

## Run locally

From the workspace root:

```bash
npm install
npm run dev:hotel
```

Or from this folder:

```bash
npm run dev
```

Open `http://localhost:3001`.

## Environment

Copy `.env.example` to `.env.local` if you want to override the local site URL.
